This project is a stable matching problem algorithm. In this scenario each owner has a list of dogs. The list is in favorite to least favorite order. Similar to the owners, each individual dog has a list of owners from favorite to least favorite. The algorithm will find the best possible matches based on each owner's and dog's preferences. Both lists of favorites are generated randomly every time you refresh the page.
